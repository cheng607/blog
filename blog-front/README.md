# 个人博客实现

#### 头部区域（Header）

- 顶部导航栏
  - 左侧：博客 Logo（支持点击返回首页）
  - 右侧：导航菜单（首页 / 归档 / 标签 / 分类 / 关于 / 搜索）
  - 特色功能：随机生成的诗句或短句作为副标题，支持打字机效果和循环显示
  - 交互：鼠标悬停时菜单选项显示下划线动画

#### 2. 主体内容（Content）

- **左侧侧边栏**：
  - **个人信息卡片**：头像（支持旋转效果）、博主名称、简介、社交链接（GitHub / 微博等）
  - **文章分类**：树形结构展示分类，支持展开 / 收起
  - **标签云**：按标签使用频率动态调整字体大小
  - **近期文章**：显示最新发布的 5-8 篇文章标题及摘要
  - **自定义组件**：可添加电子钟、历史今日、访问统计等扩展模块
- **右侧主内容区**：
  - 文章列表
    - 封面图布局：支持左右分布或居中显示，图片可链接至文章详情页
    - 文章摘要：显示标题、发布时间、分类标签及前 100 字内容
    - 交互：鼠标悬停时显示「阅读全文」按钮
  - 文章详情页
    - 顶部大图：支持自定义背景图或纯色填充
    - 文章内容：含代码高亮、行号显示、图片缩放等功能
    - 相关推荐：基于标签或内容的文章推荐模块，以卡片形式展示
    - 评论区：集成 Twikoo/Waline 等第三方评论系统，支持表情回复和点赞

#### 3. 页脚区域（Footer）

- 版权信息：© 2025 博主名称 | 备案号
- 主题信息：基于 Butterfly 主题，链接至 Hexo/Halo 官方文档
- 附加功能：可添加「友链」页面入口或「网站地图」链接

## 功能：

文章归档、文章标签、文章分类
